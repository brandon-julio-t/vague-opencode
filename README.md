## Usage

1. Copy `vague.json` to `~/.config/opencode/themes/vague.json` (or `.opencode/themes/vague.json` for a project-only theme)
2. Open OpenCode
3. Run `/theme` and pick `vague`, or set `"theme": "vague"` in your OpenCode config
